## How to Create a Test PDF

### Option 2: Use Online Converter
1. Copy this text

### Option 3: Use Command Line (Mac/Linux)
```bash
# Install pandoc

sudo apt-get install pandoc  # Linux

# Convert
pandoc SAMPLE_CONSENT_TEMPLATE.md -o sample_consent.pdf
```

Then process it:
```bash
cd scripts
```
